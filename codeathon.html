Codeathon

Event Name: Codeathon

Description: Codeathon is an exciting competition designed to test participants' programming knowledge, problem-solving abilities, and real-world application skills across three challenging rounds.

Team size: 3-4 members

Round 1: Fundamental Programming Questions (Multiple Choice). This round assesses participants' theoretical knowledge through multiple-choice questions on topics like programming concepts, algorithms, and software development practices.

Duration: 1 hour

Slots:

Slot 1: 9:00 AM - 10:00 AM

Slot 2: 10:00 AM - 11:00 AM

Round 2: Problem Solving and Algorithm Design. Participants will solve coding challenges that require creative thinking and efficient algorithm design.

Round 3: Practical Coding and Application Development. In the final round, participants will implement real-world solutions for various objectives.
